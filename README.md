## Installation

```bash
# fork and clone repo
git clone git@github.com:NotBrandX/devbranch-frontend.git

# checkout dev branch
git checkout -b dev

# install react
npm create-react-app <folder name>

# install react router
npm install react-router-dom

# install dependencies
npm install

# run app in local browser
npm run start
```
